      :type num_commands: int

      .. rubric:: Examples

      >>> import arkouda as ak
      >>> from arkouda.history import NotebookHistoryRetriever
      >>> h = NotebookHistoryRetriever()

      >>> 4*6
      >>> 2**3
      >>> h.retrieve(num_commands=3)
      ['1+2', '4*6', '2**3']
